Linear regression

Lets fit a linear model with age, gender and attitude es explanatory variables and exam points as an outcome

model1 <- lm(points~ age + gender + attitude, data = learning14)

Interpretation

Attitude is positively associated with exam points the estimate being 3.6 (when attitude increases by one, points increase by 3.6) The association is statistically significant (p-value < 0.001). Age and gender have negative, non-significant association. -> Lets remove them from the model. The estimate of attitude decreases a bit (3.5) and remains strongly significant.

Multiple R-squared of the model with all three explanatory variables is 0.2018 which means that about 20% of the variation in exam points can be explained by the three variables.

Model diagnostics

Testing assumptions of linear model.

Residuals vs fitted values

There seems not to be a pattern in the red line, suggesting it is safe to assume linear relationship between the explanatory variables and exam points outcome.

Based on QQ-plot there seems not be any strong violation to the normality assumption as there is no clear pattern.
